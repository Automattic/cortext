# Security

Cortext is in beta. Do not use it on production sites or with data you cannot afford to lose.

The data layer and access model are still moving. Early versions will not include migrations or upgrade paths.

## Reporting

Please report security issues through GitHub private vulnerability reporting when it is available. If it is not available, contact the maintainers privately before publishing details.

Do not include sensitive site data, credentials, license keys, or private content in a report.
