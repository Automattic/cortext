# Roadmap

## Phase 1: a one-month PoC

This is a wide PoC, not a deep pass on any single feature. The goal is to prove the approach end to end before polishing the details.

Baseline goals for the month:

-   A full-screen admin shell with sidebar navigation, nested pages, and collection views.
-   In-canvas content modeling and editing powered by DataForm and DataViews.
-   Cross-type taxonomies (catchy name TBD) that add fields to individual rows.

Stretch goals:

-   Basic `Formula` type.
-   Relationships between collections.
-   Frontend rendering.
-   Import from different sources.

Success metrics:

-   Social media feedback and interactions.
-   Plugin active installs on WordPress.org.
-   **Bonus**: issues or PRs filed against Gutenberg or WP Core based on gaps found during the work.

## What's next

We will decide after the PoC, based on what worked, what broke, and what people ask for.
