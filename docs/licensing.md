# Licensing

Cortext is licensed under GPL-2.0-or-later.

That is the boring choice, and that is the point. Cortext is a WordPress plugin, loads inside WordPress, and builds on WordPress and Gutenberg packages that use GPLv2-or-later licensing. Matching that license keeps distribution simple for WordPress users and for the WordPress.org plugin directory.

The license does not block paid work around the project. Hosted services, managed installs, migration help, support, templates, and other paid offerings can exist later while the plugin code remains free software.

GPLv3-or-later and AGPLv3-or-later can be revisited later. They are not part of the first open-source release. AGPL needs explicit policy review before any WordPress.org distribution plan depends on it, and a later move to AGPL would need the right contributor permissions.
