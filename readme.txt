Notion-inspired workspace on your own WordPress site: nested pages, typed collections with multiple views, and cross-type taxonomies.

== Description ==

Cortext enables a Notion-inspired workspace on your own WordPress site, perfect for creating your knowledge base. You get nested pages, typed collections with multiple views, and cross-type taxonomies that add fields to individual rows. Built on native WordPress primitives: CPTs, post meta, taxonomies, Gutenberg, DataViews, Block Bindings.

Status: prototype under active development.
